## 📁 プロジェクト構造

```
maw-app/
├── src/
│   ├── components/          # 再利用可能なコンポーネント
│   │   ├── ui/             # UIコンポーネント
│   │   ├── auth/           # 認証関連コンポーネント
│   │   └── layout/         # レイアウトコンポーネント
│   ├── pages/              # ページコンポーネント
│   ├── store/              # Zustand状態管理
│   ├── types/              # TypeScript型定義
│   ├── lib/                # ユーティリティ関数
│   └── styles/             # スタイルファイル
├── public/                 # 静的ファイル
└── package.json
```
